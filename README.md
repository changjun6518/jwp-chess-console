## jwp-chess2

## 기능 목록
- 콘솔 UI에서 체스 게임을 할 수 있는 기능을 구현한다.

## 구현 목록
-[x] piece 구현
    - 퀸, 비숍, 킹, 나이트, 룩, 폰
    - 색
-[x] 1단계는 체스 게임을 할 수 있는 체스판을 초기화한다.
    - 체스판에서 말의 위치 값은 가로 위치는 왼쪽부터 a ~ h이고, 세로는 아래부터 위로 1 ~ 8로 구현한다.
-[ ] 콘솔 UI 기능
    - 체스말, 빈칸 표시
    - white는 소문자, black은 대문자, 빈칸은 .으로 출력
-[x] 체스 말 이동 기능
    - 각 말마다 행마법 다름
    - 행마법에 따라 보드에서 움직임 기능
